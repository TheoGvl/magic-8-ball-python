# Magic 8-Ball

Μια διασκεδαστική, ψηφιακή εκδοχή της κλασικής (Magic 8-Ball) που απαντάει σε όλες σου τις ερωτήσεις! Το project αυτό δημιουργήθηκε για να εξασκηθώ στη χρήση (Lists) και στην ενσωμάτωση της τυχαιότητας (`random`) σε ένα μοντέρνο Γραφικό Περιβάλλον (GUI) με τη βιβλιοθήκη Flet.

## Τι ακριβώς κάνει;
* Ο χρήστης πληκτρολογεί μια ερώτηση που απαντιέται με "Ναι" ή "Όχι".
* Πατώντας το κουμπί **«Ρώτα τη Μπάλα!»**, το πρόγραμμα επιλέγει και εμφανίζει εντελώς τυχαία μια από τις προκαθορισμένες μαντεψιές (π.χ. "Είναι απολύτως βέβαιο!", "Ρώτα ξανά αργότερα...").
* **Έξυπνος Έλεγχος:** Αν ο χρήστης πατήσει το κουμπί χωρίς να έχει ρωτήσει τίποτα, η εφαρμογή το καταλαβαίνει και του πετάει κόκκινο μήνυμα σφάλματος!

**Βασικές γνώσεις που εφαρμόστηκαν:**
1. **Λίστες (Python Lists):** Ομαδοποίηση πολλών διαφορετικών κειμένων (απαντήσεων) σε μία μόνο μεταβλητή.
2. **Η συνάρτηση `random.choice()`:** Επιλογή ενός τυχαίου στοιχείου μέσα από μια λίστα.
3. **Δυναμικό UI (State Management):** Αλλαγή όχι μόνο του κειμένου, αλλά και του χρώματος του αποτελέσματος σε πραγματικό χρόνο, ανάλογα με το αν υπήρξε σφάλμα ή επιτυχής μαντεψιά.
